AIwrite is here!

Many people use AI to help them write documents particularly professional documents, but using standard online tools for writing is too cumbersome involving many copy/paste operations and tend to generate texts that are to generic and cannot be used directly before extensive manual editing. In order to help is this type of task, we have developed AIWrite, a very simple text generator using your choice of AI agent, that not only lets you properly and easily specify the structure of technical documents, but also makes it very easy to provide the AI information of your specific area of expertise, in the form of a collection of reference documents, so that the AI gent will write like a pro, not like a newbie!

New RegDBot Release: v0.6.0

We have just release a new version of Reggie D. Bot, v0.6.0. This release includes the following features: Improved user interface, with a new design and better user experience. Log of code snippets: Now every code snippet generated by Reggie D. Bot is saved in a log, so that you can easily access it later. The log entry is stored alongside the question that motivated it and an explanation. Many bugfixes and performance improvements.

Demo of Reggie D. Bot is online

After a few months of hard work, we are pleased to announce that we have made the first release of Reggie D. Bot. The demo is available at regdbot.deeplearn.ltd. It is still an alpha release, So at this point you can’t upload new datasets or register as an user. This release is for testing purposes only, and is linked with our project in the Backdrop build v5 program. Next steps before having a beta release, are to add the ability to upload new datasets, and to register as an user.
