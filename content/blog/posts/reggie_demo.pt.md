Após alguns meses de trabalho árduo, temos o prazer de anunciar que lançamos o primeiro lançamento do Reggie D. Bot. A demonstração está disponível em [regdbot.deeplearn.ltd](https://regdbot.deeplearn.ltd). Ainda é uma versão alfa, portanto, neste momento, você não pode enviar novos conjuntos de dados nem se registrar como usuário. Esta versão é apenas para fins de teste e está vinculada ao nosso projeto no programa [Backdrop build v5](https://backdropbuild.com/builds/v5/reggie-d-bot-yq9d).

Os próximos passos antes do lançamento beta são adicionar a capacidade de enviar novos conjuntos de dados e se registrar como usuário. Também estamos trabalhando para incluir chamadas de função ao agente de IA, para que ele também possa gerar gráficos com base nos resultados da consulta.

Fique ligado para mais atualizações sobre este projeto!
