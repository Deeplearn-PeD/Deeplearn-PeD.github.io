# Geração Aprimorada por busca de dados

## Resumo
Uma aplicação popular em IA é a geração de textos e produtos de análise de dados com base no conhecimento existente. Esta é uma tarefa desafiadora, pois exige que o modelo compreenda o contexto dos dados de entrada e gere uma saída significativa. Dados estruturados, como bancos de dados relacionais, e dados não estruturados, como texto, imagens e vídeos, são usados ​​como contexto para os modelos generativos.

Na DeepLearn, chamamos essa tarefa de *Geração Aprimorada de Recuperação (REG)*. Desenvolvemos uma nova abordagem para REG que combina os pontos fortes dos modelos generativos e baseados em recuperação, aprimorando-os com um processo baseado em IA que facilita a elicitação das necessidades analíticas dos usuários.

## Líder da equipe
- [Reggie D. Bot](/team/REG.md) (Líder da equipe) Reggie é um bot de IA especializado na geração de textos e produtos de análise de dados com base no conhecimento existente. Reggie orquestra diversos modelos de IA para gerar a melhor saída possível para o usuário. Reggie é um modelo de IA de última geração que está em constante aprendizado e aprimoramento.
